使用VMware安装CentOS

Created2022-01-19|Updated2022-01-20
|Post View:

一、准备工作

VMware为虚拟机程序、后者为一免费且简单的Linux系统。

请阁下购买VMware程序且激活后再往下进行

CentOS的下载连接为：www.centos.org

请阁下注意您的处理其架构为何，如X86/ARM

请阁下下载代替有dvd字样的*.iso文件

请阁下注意最新的CentOS 8已不提供32位版本

二、安装方法

新建虚拟机-自定义（高级）-稍后安装操作系统-Linux（版本CentOS 7 64位）-位置（建议改为D盘）-每个处理器的内核数量4或8（看个人电脑配置而定）-虚拟机的内存4G（小体系）-使用网络地址转换-LSIlogic-SCSI-创建新虚拟磁盘-将虚拟磁盘储存为单个文件（建议80GB）-完成-CD/DVD-使用ISO镜像文件（在CentOS官网下载的）-开机（绿色按钮）-INSTALL CENTOS 7-ENGLISH-DATE&TIME（CHINA）-language support(中文)-SOFTWARE SELECTION-GNOME Desktop(CentOS 8 中已经没有,请略过该步骤与下一步)-GNOME APPLICATION & LEGACY X WINDOW SYSTEM COMPATIBILITY & COMPATIBILITY LIBRARIES & DEVELOPMENT TOOLS -NETWORK&HOSTNAME-OPEN（常开网络）-ROOT PASSWORD(设置root用户账号密码)-USER CREATION（设置用户账号密码）-GEGIN INSTALLATION-等待一段时间-I accept the license

三、其它问题

1、建议安装的系统语言为ENGLISH，以避免不必要的麻烦

2、CentOS 8 已经取消GNOME Desktop图形化界面，因此阁下安装CenOS 8 完成后，需要自行搭建环境，否则安装应用程序时可能会报错。

搭建环境时请使用root账户登陆、并保持网络畅通

安装gcc：yum install gcc

安装g++：yum install gcc-c++

安装gdb：yum install gbd

若出现：error: unable to find a match: python-devel

例如下边这个例子会出现该问题：

git clone –recursive –shallow-submodules git://github.com/usnistgov/COSMOSAC

cd COSMOSAC

python setup.py install

解决方案为：yum install python36-devel
